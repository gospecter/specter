# SpecterSync DIY vs SpecterSync Pro

SpecterSync has two editions with different goals.

**SpecterSync DIY** is the open-source self-build edition for developers and technical operators. It is licensed under AGPLv3 and focuses on direct local sync.

**SpecterSync Pro** is the official commercial product. It adds signed installers, automatic updates, license activation, managed connector services, hosted AI transformation workflows, support, and commercial operations.

| Capability | SpecterSync DIY | SpecterSync Pro |
|---|---:|---:|
| Source available | Yes | Open-source components only |
| License | AGPLv3 | Commercial EULA |
| Self-build | Yes | Not required |
| Direct local CMS sync | Yes | Yes |
| Local dry-run preview | Yes | Yes |
| Official signed macOS app | No | Yes |
| Official Windows/Linux installers | No | Planned/Yes |
| Automatic updates | Manual pull/build | Yes |
| License activation | No | Yes |
| Managed Shopify OAuth/refresh | No | Yes |
| Hosted AI transformation workflows | No | Yes |
| CMS-aware safe apply checks | Basic/local | Advanced/service-backed |
| Support | Community/best effort | Included |
| Commercial invoices/refunds | No | Yes |

## Positioning

Community is for transparency, self-building, learning, and contribution.

Pro is for people and teams who want SpecterSync maintained, signed, updated, supported, and ready for production work.

The paid Pro value is not permission to sync. The paid value is the production lane: managed connectors, support, automatic updates, and hosted AI transformations that understand CMS constraints.
