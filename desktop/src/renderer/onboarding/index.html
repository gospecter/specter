Welcome to Specter

Two-way sync between your Ghost blog and a folder of markdown files. We'll get you set up in three quick steps.

Connect to Ghost

Find your Admin API key under Ghost Admin → Settings → Integrations → Add custom integration.

Pick the sync folder

Choose the local folder where your Ghost posts should live. Specter watches this folder for changes and keeps it in sync with your blog.

No folder chosen

Tip: this can be any folder. If you use Obsidian, point it at a folder inside your vault.

✓

Ready to sync

Specter will run quietly in your system tray. The first full sync starts as soon as you finish setup.

  • Auto-push when you edit files in the sync folder
  • Periodic pull every 10 minutes for remote changes
  • Conflict prompts when both sides change the same post
